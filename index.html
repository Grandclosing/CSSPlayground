:before and :after pseudo-selectors

The above text is created using CSS's :before and :after pseudo-selectors.

Those letters are not select-able because they literally don't exist in the DOM. Well, they do, and they don't.

I've found this to be a really good resource on this topic.


Basically, :before injects stuff before the actual content of the element, and :after does it after - but both pieces of content are still confined to / bound to the element itself.

These pseudo-selectors take the CSS content property, which can take a string, an image url, nothing, or a counter

CSS counters

I did NOT know that this was possible, but here we are... CSS has counter logic. This can help with numbering lists.

In fact, this page is using counters. The "Section #:" parts of the titles use a counter named "section".

To create a counter, first declare/initialize it with a counter-reset property:

body {
  counter-reset: section;
}

Then, for the tags, classes, ids, etc. that rely on the counter, increment it, and display it via content:

h1:before {
  counter-increment: section;   
  content: "Section " counter(section) ": '";
}

It seems CSS counters are naturally suited for Table of Contents styling:

  1. Introduction
  2. Installation
    1. Pre-requisites
    2. Troubleshooting
    3. Configuration
      1. Configuration File
      2. Environment Variables
      3. Some other topic...
    4. Usage
  3. Glossary
  4. Terms

The code for that: 

ol {
    counter-reset: item;
    list-style-type: none;
}

li::before {
    counter-increment: item;
    content: counters(item, ".") " ";
    color:#3aadff;
}

CSS Pseudo-elements vs Pseudo-classes

I was always confused about the appearance of single colons, and double colons in CSS code... until now.

Double colons were introduced to distinguish pseudo-elements from pseudo-classes which use single colons (for historical and backwards-compatibility.)

According to W3-Schools:

"A CSS pseudo-element is used to style specified parts of an element.

For example, it can be used to:
- Style the first letter, or line of an element
- Insert content before, or after, the content of an element"

Some common pseudo-elements are ::first-line, ::first-letter, and ::selection:
